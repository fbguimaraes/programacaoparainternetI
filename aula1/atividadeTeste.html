O que é HTML ?

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar páginas da web. Ele estrutura o conteúdo da página, dizendo ao navegador como exibir textos, imagens, links, vídeos, formulários e outros elementos.

Curiosidades sobre o html

  • HTML foi criado nos anos 90

    • O HTML foi desenvolvido por Tim Berners-Lee em 1991, enquanto trabalhava no CERN. Ele também criou o primeiro navegador e servidor web!

  • A primeira versão tinha apenas 18 tags

    • Hoje temos mais de 100 tags disponíveis no HTML5, mas a primeira versão do HTML era bem simples, com poucas opções para estruturar páginas.

  • HTML não é uma linguagem de programação

    • Apesar de ser essencial para a web, o HTML não executa lógica ou cálculos — ele apenas organiza o conteúdo.
